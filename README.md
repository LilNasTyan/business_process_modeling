Это учебный проект по дисциплине "Моделирование бизнес процессов".

Здесь реализованы 3 алгоритма построения модели процесса (Alfa, Inductive, Heuristics)
и проведён анализ качественных метрик для полученных моделей (precision, fitness, generalization)   .

Построение моделей процесса основано на тестовых логах, которые предоставляет сама библиотека PM4PY в своём репозитории.
